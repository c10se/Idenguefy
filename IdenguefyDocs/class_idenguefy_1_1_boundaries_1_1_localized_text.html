Detailed Description

Translates the language of relevant text fields selected within Unity's environment to a language selected from a list of options by the user.

This component finds all TMP_Text objects and registers them for translation, skipping any with an "IgnoreLocalisation" tag. It also watches for new text elements added at runtime.
