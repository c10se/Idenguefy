Detailed Description

Responsible for rendering the saved map pointers additional UI interfaces.

Responsibilities:

  1. Displays respective popup windows for Creating/Editing/Deleting a pointer
  2. Displays confirmation for deleting a pointer
  3. Displays a list of all pointers created by the user

Author: Cheuk Hong Version: 1.0
