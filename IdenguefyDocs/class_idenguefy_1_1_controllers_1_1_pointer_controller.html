Detailed Description

Manages user-created map pointers (e.g., saved or tagged locations).

Provides full CRUD functionality:

  1. CreatePointer: Add new pointer
  2. ReadPointer: Retrieve pointer by ID
  3. EditPointer: Modify details of an existing pointer

  4. DeletePointer: Remove pointer from the list

    Author: Cheuk Hong, Sharina, Napatr, Xavier Version: 1.0 Notes: N/A
