Detailed Description

Responsible for rendering the users saved map pointers onto the UI map.

Responsibilities:

  1. Convert pointer coordinates (lon/lat) to pixel positions through MapUtils Class
  2. Assigns the correct Pointer Sprite to the pointer based on its home tag
  3. Showcases information that the user has written about the pointer in a dedicated UI popup window
  4. Includes buttons as means to access additional functionalities like editing/deleting the specified pointer

Author: Cheuk Hong Version: 1.0
