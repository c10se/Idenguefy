Detailed Description

Utility class providing coordinate conversion functions for the map system.

Converts longitude/latitude into tile indices or pixel ratios according to the Web Mercator projection.

Converts geographic coordinates (longitude, latitude) into a pixel ratio relative to a minimum tile boundary.

This is used to calculate the precise position of a point within the map panel, offset by the top-left tile (xMin, yMin).

Parameters
longitudeThe geographic longitude coordinate.
latitudeThe geographic latitude coordinate.
zoomThe current map zoom level.
xMinThe minimum X tile index (left-most tile) of the map boundary.
yMinThe minimum Y tile index (top-most tile) of the map boundary.
xPixelRatioOutput: The calculated X position as a tile ratio, offset from xMin.
yPixelRatioOutput: The calculated Y position as a tile ratio, offset from yMin.
