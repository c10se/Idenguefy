Detailed Description

The SearchAPIHandler implementation for MapTiler's Geocoding API.

Responsibilities:

  1. Request relevant GeocodingAPI location data based on the user's search query
  2. Download the raw Geocoding dataset
  3. Parse and convert features into MapSearchResult entities
  4. Add preprocessed search results into an array

  5. Wrap array with appropriate SearchAPIResponse Wrapper
