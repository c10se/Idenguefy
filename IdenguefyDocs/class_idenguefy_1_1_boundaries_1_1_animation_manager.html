Detailed Description

AnimationManager is where all animation related logic is kept in:

  1. Holds all the animation methods that are used within the unity environment
  2. Holds all animators created and used as well as the properties required in each animation

.

Author: Xavier Version: 1.0 Notes: This is a singleton that persists across scenes.
