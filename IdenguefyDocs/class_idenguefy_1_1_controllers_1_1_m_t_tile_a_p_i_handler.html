Detailed Description

The SearchAPIHandler implementation for MapTiler's Geocoding API.

Responsibilities:

  1. Request relevant tile (x, y, zoom) data to obtain texture data
  2. Download the given raw 2d texture of the raster tiles
  3. Parse and convert features into Texture2D entity

  4. Wrap array with appropriate TileAPIResponse Wrapper
