Detailed Description

A generic data manager for saving and loading lists of serializable data using PlayerPrefs.

It wraps the list in a container class to allow for JSON serialization. Works for pointers, settings, or any other [System.Serializable] class.

Template Parameters
TThe type of data being managed (e.g., MapPointer, Settings).
