Detailed Description

Used to update all applicable objects colour depending on the users selected colour theme.

This component finds all UI elements in the scene and applies theme colors. It also swaps sprites based on a "Light Mode" / "Dark Mode" naming convention.

Author: Napatr Version: 1.0
