Detailed Description

Handles fetching and preprocessing of preventive health tips.

Responsibilities:

  1. Takes in a list of strings for helping to prevent Dengue
  2. Convert the strings to Health Tip objects
  3. Store these into a list of Health Tip object called Tips Provides a public Tips list accessible by PreventiveHealthTipView
