Detailed Description

Responsible for rendering dengue clusters on the UI map as colored circles.

Responsibilities:

  1. Convert cluster coordinates (lon/lat) to pixel positions through MapUtils Class
  2. Compute approximate cluster radius from polygon points through MapUtils Class
  3. Create semi-transparent colored circle UI elements inside mapPanelRect
  4. Color code clusters by severity
  5. Shows information about the cluster in a dedicated UI popup window

Adjusts for map panel offset and pivot alignment to ensure accurate placement Creates a color coded cluster based on serverity
