Detailed Description

Handles the overall flow of map rendering, tile management, and dengue cluster visualization.

Workflow Overview:

  1. Compute map tile boundaries based on Singapore’s geographical coordinates.
  2. Sequentially load and render all map tiles from the MapTiler API.
  3. Cache downloaded tiles to improve performance and reduce API calls.
  4. Ensure all tiles are loaded before rendering dengue cluster overlays.
  5. Integrate with ClusterController and ClusterCircleDrawer to display real-time cluster data.

Author: Eben, Xavier
Version: 2.0
Note: Map tiles are capped at zoom level 15 to avoid API quota errors.
