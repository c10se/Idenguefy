Detailed Description

Manages the disk-based caching of map tiles.

Tiles are compressed using GZip to save space and are stored in a platform-specific persistent data path. Includes thread-safe mechanisms to prevent reading a tile while it is being written.

Author: Eben, Ryan Version: 1.0
