Detailed Description

The main interface in which the listed UI objects are loaded onto, in order:

  1. Map Tiles
  2. Clusters
  3. Saved Pointers (If any)
  4. Live Location

Includes functionality to recenter back onto the users live location or a searched location Includes a toggleable function to read the users touch input to detect when a user is pressing and holding down on the screen.

Author: Eben, Sharina, Cheuk Hong, Xavier Version: 1.2
