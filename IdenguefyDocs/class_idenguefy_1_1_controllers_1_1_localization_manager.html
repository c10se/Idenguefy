Detailed Description

Manages runtime text localization for the Idenguefy application.

Responsibilities:

  1. Registers UI elements and updates their language dynamically.
  2. Communicates with Microsoft Translator API for translation.
  3. Applies throttling, caching, and retry logic to prevent rate limits.
  4. Persists the current language state globally across scenes.

Author: Napatr
Version: 1.0
This controller ensures consistent and efficient translation handling throughout the app.
