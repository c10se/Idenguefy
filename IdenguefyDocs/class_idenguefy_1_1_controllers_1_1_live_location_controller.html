Detailed Description

Manages live GPS location data for the Idenguefy application.

Responsibilities:

  1. Retrieves the user’s real-time GPS coordinates using Unity’s location services.
  2. Provides mock location support for testing within the Unity Editor.
  3. Exposes a public interface to access the user’s current location and readiness state.

Author: Xavier Version: 1.0
This controller provides a single point of access for location-based data, ensuring that other systems (e.g., AlertManager, Map utilities) can safely query the user’s coordinates.
